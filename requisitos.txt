Requisitos Funcionais

[x] O usuário deve poder criar uma nova transação;
[x] O usuário deve poder obter um resumo da sua conta (transações);
[x] O Usuário deve poder listar todas as transações que já ocorreram
[x] Usuário deve poder visualizar uma transação única

Regras de Negócios

[x] A transação pode ser do tipo crédito que somará ao valor total ou débito que subtrairá
[ ] Deve ser possível identificar os usuários entre as requisições
[ ] O usuário só pode acessar transações que ele mesmo criou
